# 🎧 **>B1ueMusic<流批一体数据处理平台** 🎧

## 🎶项目背景
  随着音乐平台数据体量的提升，传统离线处理模式已无法满足实时分析与多维监控的需求。本项目设计并实现一套支持日志+业务数据统一采集、处理、存储、展示的流批一体数据处理体系，以支撑用户行为分析、平台运营监控等核心功能。
